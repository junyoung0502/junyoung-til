### 새로 배운 내용
#### 타입 힌트: 정의, 사용 이유, 사용 방법 등

- 정의: 변수, 함수의 인자, 반환값의 형식을 명시적으로 표시하는 문법
- 사용 이유: 프로젝트 진행시 타입을 명확하게 파악하여, 유지보수 및 협업에 도움을 주기 위함

- 사용 방법: ':(클론)'과 '->(화살표)'를 사용해 타입을 입력
'''python
# 예시 코드
def input_function(i_list: List[str], item_list: List[str]) -> str:
# def input_function(i_list, item_list):

    while True:
        # 사용자 입력
        user_input = input("\n 번호를 입력해주세요:")

        if user_input in i_list:
            # 선택한 번호와 풋살장 매칭
            select_item = item_list[int(user_input)-1]
            print(f"\n {select_item}을 선택하셨습니다.")
            return select_item
        else:
            print("\n 잘못된 입력입니다. 다시 입력하세요.")
            continue
'''

- 참고사항
    - 기존 정적 언어에서는 타입 힌트 작성으로 타입을 강제하거나 실행 흐름을 바꾸는 것이 가능하지만, 파이썬은 동적 타입 언어로 타입 힌트가 실행 과정 개입하지 않는다.
    타입 힌트와 다른 매개변수가 전달되어도 인터프리터는 이를 오류로 처리하지 않고 그대로 실행한다.
    타입 힌트는 부가적인 정보로 코드의 의도를 명확히 드러내는데 사용이 된다.

    함수 인자에서 타입 힌트와 기본값을 같이 쓰기

        2. 당시 코드
            '''
            # 기본값 할당(= None)을 타입 힌트(: Optional)보다 먼저 작성함 -> 에러 발생
            def show_list(data_list: List[str], price_dict=None: Optional[Dict[str, str]]) -> List[str]:

            '''
        
        3. 해결 방법
